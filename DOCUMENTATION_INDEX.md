## 🔧 CUSTOMIZATION GUIDE

### Change Colors
1. Open any `.blade.php` file
2. Find `@push('styles')` section
3. Update CSS variables or values
4. Save and refresh

Example:
```css
.btn-primary { background: #YOUR_COLOR; }
```

### Add More Mock Data
1. Locate `@php` section in file
2. Duplicate array item in `$mockProducts`/`$mockArticles`/etc
3. Update the values
4. Add corresponding HTML loop item

Example:
```blade
@php
    $products = [
        ['id' => 1, ...existing...],
        ['id' => 7, ...new item...],  // Add here
    ];
@endphp
```

### Change Text/Labels
1. Find the text in the Blade file
2. Search for the string
3. Replace with new text
4. Save and refresh

Example:
```blade
<h1>{{ 'Katalog Produk' }}</h1>  <!-- Change text here -->
```
