🔧 DATABASE SCHEMA

Consultations Table (13 columns):
  id                 bigint (PK, auto-increment)
  user_id            bigint (FK to users, nullable)
  skin_story         text (user's skin description)
  tags               json (manual tags array)
  detected_traits    json (AI-detected traits)
  concern_1          varchar(50) (primary concern)
  concern_2          varchar(50) (secondary concern)
  preferences        json (vegan, fragrance-free, etc)
  recommendations    json (future: AI recommendations)
  status             enum (pending/processed/archived)
  created_at         timestamp
  updated_at         timestamp

Indexes:
  ✅ Primary key: id
  ✅ Foreign key: user_id (cascade delete)
  ✅ Index on: status
  ✅ Index on: user_id
  ✅ Full-text index on: skin_story

🚀 READY FOR DEPLOYMENT

Setup Steps (for deployment):
  1. Configure .env with MySQL credentials
  2. Create database: CREATE DATABASE skinquo;
  3. Run migrations: php artisan migrate --force
  4. Clear caches: php artisan config:clear && php artisan route:clear
  5. Start server: php artisan serve
  6. Test: http://localhost:8000/consultation
